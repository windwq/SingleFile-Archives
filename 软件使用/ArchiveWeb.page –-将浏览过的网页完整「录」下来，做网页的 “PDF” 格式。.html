ArchiveWeb.page – 将浏览过的网页完整「录」下来，做网页的 “PDF” 格式。

Chrome  2022/09/19  青小蛙  11

ArchiveWeb.page 是一款可以将浏览过的网页完整的录下来的 Chrome 扩展，可保存下高保真度的网页存档，包括外部资源，使用起来就像联网使用一样。@Appinn

青小蛙专门对比测试了一下与 Save Page 等类似扩展的区别（同时访问发现频道这个页面），ArchiveWeb.page 能够实现包括页面按钮、菜单等一些交互的完整还原，但其他工具仅保存了简单的 HTML 页面，很多页面交互实效，无法使用。

当然还有一个区别，就是如果你在录制的时候访问了多个页面，那么 ArchiveWeb.page 也会帮你录下来，再未来使用时，同样是可以点击并浏览的。

来自发现频道，@jerrylus 同学的推荐:

ArchiveWeb.page – 将浏览的网页「录」下来

若是想将单个网页存档，最简单的方式可以打印当前页面到 PDF，复杂一些可以用 Save Page / SingleFile 之类的插件，嫌麻烦也能直接提交链接到 archive.org。但倘若是想要存档多个页面，或是保留页面的可交互性，或是存档登陆后才可访问的内容，以上的方式或多或少都有些问题。

ArchiveWeb.page 的解决思路则很简单：点击「录制」按钮，随后正常浏览需要被存档的网页。浏览过程中，这个插件会把浏览过的页面和页面加载的外部资源一并存储下来。浏览完成后点击「结束录制」，就得到了一份浏览过的网页们的完整本地副本。不仅页面完全被保存（甚至连页面上的 Javascript 也能正常运行），点击链接在页面间跳转时，也会自动从副本中读取（如果下一个页面被存档了的话）。

开始录制

实现上，这一插件使用了 Chrome 系浏览器的调试机制来获取外链资源，所以 Chrome 系列的浏览器应该都能使用。存储下来的网页不仅可以用插件在进行录制的浏览器中访问，更可以导出成通用的 WACZ (Web Archive Collection Zip) 或 WARC (Web ARChive) 文件，在其他工具中使用/回放（如 ReplayWeb.page）。

后台管理界面

在 ArchiveWeb.page 中，用 Page 来代表一个录制集，你可以多次将不同的网页录制进一个 Page 中，然后会在后台看到每一个 Page 中的不同网页，点进去之后，会打开一个浏览器中的浏览器。
