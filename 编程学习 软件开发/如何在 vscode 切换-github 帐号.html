如何在 vscode 切换 github 帐号

来自 <https://jing-siao.github.io/changeGitAccountInVscode/>

 

假设你有一个自己的 github 帐户和公司的 github 帐户，因为同一台电脑，每次上传都只用自己的 github 帐户上传，两个又不想要混一起用，那怎么办呢 ?

安装套件 git-autoconfig

git-autoconfig

设定 github 帐号

ctrl+p 先叫出搜寻视窗，在视窗输入 >git 找到 git-autoconfig:Set Config

Set Config

之后输入 email 跟 name

输入email

输入name

好了之后可以在 settings.json(File > Preferences > Setting) 看到刚刚新增的，找不到 settings.json 的，右上方有一个文件旋转的按钮 open settings(JSON) ，点下去就可以看到了

settings

然后再重复刚刚的动作新增第二个帐号，你也可以在 settings.json 里自己输入

切换 github 帐号

设定好后，要来 push 上去了，假设你今天要用另一个帐号 push ，作法一样 ctrl+p 先叫出搜寻视窗，在视窗输入 >git 找到 git-autoconfig:Set Config ，这时候可以选择你要用哪一个帐号上传

切换帐号

按下 Enter 键后，就可以 push 了，在 github commit 上面看就会是不同帐号啰 ! 是不是很简单呀 ~
